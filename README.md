## 🚀 Desenvolvimento

Este projeto é desenvolvido usando:

- Godot Engine 4.2
- GDScript
- Laravel v10.46.0 (API)
- PHP v8.2.22 (Backend)

### Pré-requisitos

- Godot Engine 4.2 ou superior
- Android SDK (para builds Android)
- Export Templates do Godot 4.2
